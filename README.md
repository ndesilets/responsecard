<header>Turning Technologies ResponseCard Answer Sniffer</header>

Required:
-Arduino of some sort (I used a Teensy 3.1)
-nRF24L01 transceiver

What it Does:
-Records answers emitted from clickers and updates a list of all possible answers with their respective count
